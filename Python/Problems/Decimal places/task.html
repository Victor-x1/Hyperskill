Decimal places Open on JetBrains Academy

Round the number from input to the required number of decimals.

The input format:

Two lines: the first with a floating-point number, the second with an integer representing the decimal count.

The output format:

The rounded number.

Do NOT forget to convert the input numbers and to enclose each value in {} in a formatted string.

We encourage you to use string formatting, since solutions with round() function may not pass the tests. Besides, this is exactly what the topic is about!


Sample Input:
2.71828
3
Sample Output:
2.718
