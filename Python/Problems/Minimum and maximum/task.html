Minimum and maximum Open on JetBrains Academy

Imagine that your friend asked you to write a program that finds the minimum and the maximum.

Write the code that receives two integers as its input, each number goes on a new line. The output should show:

  1. The biggest number in the first line
  2. The smallest number in the second line.

Note that your friend might insert identical numbers! In this case, just output both given numbers on separate lines.


Sample Input:
2
8
Sample Output:
8
2

Sample Input:
23
23
Sample Output:
23
23

Sample Input:
34
12
Sample Output:
34
12


Memory limit: 256 MB
Time limit: 15 seconds
